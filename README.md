# Svelte 5 Pomodoro

A minimalist productivity timer built to master the new **Svelte 5 Runes** architecture.
The goal is a clean, type-safe UI that handles work and break cycles using high-performance, fine-grained reactivity.

## Getting Started

> [!NOTE]
> This project uses **pnpm**. If you don't have it installed, run: `npm install -g pnpm`

1. Clone this repo and navigate into the new directory

2. Install the dependencies:

```sh
 pnpm install
```

3. Start the development server:

```sh
 pnpm run dev
```

4. To Build the app for production:

```sh
 pnpm run build
```
